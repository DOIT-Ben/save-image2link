# 2026-05-22 公开版 exe 封装

## 目标

把现有剪贴板图片保存脚本封装为可公开发布的 Windows 小工具，支持可视化配置、右键菜单安装/卸载、用户自定义保存目录，并去除所有个人路径和本机习惯。

## 边界

- 保留核心能力：读取剪贴板图片，保存为 PNG，复制路径或图片引用。
- 默认目录改为当前用户图片目录下的 `SaveImageToLink`。
- 配置保存到当前用户应用数据目录，不随仓库发布个人配置。
- 右键菜单写入当前用户注册表，不要求管理员权限。
- v1 不接入远程图床，不生成公网链接。

## 成功标准

- 测试覆盖默认目录、配置读写、复制格式、保存逻辑和右键命令生成。
- README 说明一键安装、卸载、设置路径和打包方式。
- 仓库不提交缓存、运行结果、构建产物和个人配置。
