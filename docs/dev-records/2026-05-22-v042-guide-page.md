# 2026-05-22 v0.4.2 新手引导页

## 目标

用户反馈 `image` 和三种复制格式不够直观，需要一个详细、一步步的引导页面，而不是把说明堆在主窗口里。

## 改动

- 主窗口右上角新增 `引导` 入口。
- 新增可滚动引导窗口，分 8 步解释：
  - 先复制图片；
  - 保存目录；
  - `image` 文件名前缀；
  - Markdown 图片、纯路径、file URI 三种复制格式；
  - 安装右键菜单；
  - 实际保存流程；
  - 测试保存；
  - 常见问题。
- 新增 `filename_preview()`、`copy_format_examples()`、`guide_sections()` 作为可测试内容源。
- 更新中英文 README、Quickstart 和根说明文件。
