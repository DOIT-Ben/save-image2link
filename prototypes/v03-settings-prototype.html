保存图片链接 / SaveImageToLink

SaveImageToLink

把剪贴板图片变成可粘贴的本地引用。

右键菜单未安装
Context menu is not installed
  1. 1选择保存图片的位置，不再乱放到临时目录。
  2. 2选择复制格式，默认推荐 Markdown 图片。
  3. 3安装右键菜单，之后在文件夹空白处右键使用。
  4. 4复制截图后测试保存，确认路径可用。
本工具只在本机保存图片。不会上传，不连接图床，不内置个人路径。

安装与设置

一次设置，之后只需要在资源管理器右键保存图片。界面只保留和完成任务有关的选项。

v0.3 原型
Markdown 图片

推荐。适合 Obsidian、Markdown 文档和笔记。

纯路径

适合脚本、本机文件管理和快速定位图片。

文件链接 file://

适合浏览器、HTML 和支持标准文件链接的工具。

等待复制图片或截图。安装后，在资源管理器文件夹空白处右键选择“保存图片并复制链接”。
